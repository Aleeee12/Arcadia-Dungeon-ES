2
Creation + Settings
▼

2.1 - Creer donjons

/arcadia_dungeon admin create test_facile Donjon Facile
/arcadia_dungeon admin create test_moyen Donjon Moyen
/arcadia_dungeon admin setspawn test_facile
  • Donjons crees

2.2 - Settings complets

/arcadia_dungeon admin settings test_facile maxplayers 6
/arcadia_dungeon admin settings test_facile minplayers 1
/arcadia_dungeon admin settings test_facile maxdeaths 3
/arcadia_dungeon admin settings test_facile recruitment 90
/arcadia_dungeon admin settings test_facile timelimit 600
/arcadia_dungeon admin settings test_facile pvp false
/arcadia_dungeon admin settings test_facile teleportback true
  • Boolean: seuls true/false suggeres
  • Tous les settings fonctionnent

2.3 - Timer Warnings NEW

/arcadia_dungeon admin timerwarnings test_facile add 120
/arcadia_dungeon admin timerwarnings test_facile add 15
/arcadia_dungeon admin timerwarnings test_facile list
/arcadia_dungeon admin timerwarnings test_facile remove 120
  • Warnings ajoutes/retires/listes
3
Wand de selection + Zone NEW
▼

3.1 - Obtenir la wand

/arcadia_dungeon admin wand
  • Pelle en or recue avec nom "arcadia_wand"
  • Liste des donjons cliquables dans le chat

3.2 - Selectionner la zone

Clic gauche sur coin 1, clic droit sur coin 2 :

  • Clic gauche → "Pos1 definie: X, Y, Z" en vert
  • Clic droit → "Pos2 definie: X, Y, Z" en cyan
  • "Zone sauvegardee!" avec dimensions en or

3.3 - Changer de donjon

/arcadia_dungeon admin wand_select test_moyen
  • Donjon change, pos1/pos2 reset

3.4 - Anti-parasites + containment

  • Joueur non-participant dans zone → TP au /spawn
  • Mobs existants clear au demarrage
  • Joueur en donjon sort de la zone → TP retour au spawn donjon
  • Admin en spectator PAS teleporte
4
Boss : equip, phases, effets, commandes
▼

4.1 - Creer + equiper boss

/arcadia_dungeon admin boss add test_facile boss1 minecraft:zombie 100 8
/arcadia_dungeon admin boss setname test_facile boss1 Roi des Morts-Vivants
/arcadia_dungeon admin boss setspawn test_facile boss1
/arcadia_dungeon admin boss equip test_facile boss1 mainhand minecraft:diamond_sword
/arcadia_dungeon admin boss copyequip test_facile boss1
  • Boss equipe (commande directe + copyequip)

4.2 - Boss optionnel NEW

Dans le JSON, ajouter un 2e boss avec "optional": true, "spawnChance": 0.3

  • Boss optionnel: 30% chance, message si pas apparu

4.3 - Phases avec effets NEW

/arcadia_dungeon admin boss phase add test_facile boss1 2 0.5
/arcadia_dungeon admin boss phase set test_facile boss1 2 damage 1.8
/arcadia_dungeon admin boss phase set test_facile boss1 2 invulnerable 2
/arcadia_dungeon admin phaseeffect test_facile boss1 2 minecraft:slowness 10 1
/arcadia_dungeon admin phaseeffect test_facile boss1 2 minecraft:darkness 5 0
  • Joueurs recoivent Slowness II + Darkness quand phase 2 se declenche

4.4 - Commandes de phase NEW

/arcadia_dungeon admin phasecommand test_facile boss1 2 say Le boss entre en phase 2!
/arcadia_dungeon admin boss phase add test_facile boss1 3 0.2
/arcadia_dungeon admin boss phase set test_facile boss1 3 action KILL_SUMMONS
/arcadia_dungeon admin boss phase summon test_facile boss1 3 minecraft:skeleton 4
/arcadia_dungeon admin boss phase summonequip test_facile boss1 3 0 mainhand minecraft:iron_sword
  • Commande executee + sbires equipes

4.5 - Recompenses

/arcadia_dungeon admin reward add test_facile boss1 minecraft:diamond 3 0.8
/arcadia_dungeon admin reward completion test_facile minecraft:experience_bottle 5 1.0
  • OK

6
Progression + Anti-monopole
▼

6.1 - Prerequis

/arcadia_dungeon admin order test_facile 1
/arcadia_dungeon admin order test_moyen 2
/arcadia_dungeon admin require test_moyen test_facile
/arcadia_dungeon progression
  • Facile dispo, Moyen verrouille, cliquable

6.2 - Admin progression

/arcadia_dungeon admin progress set PSEUDO test_facile 5 120
/arcadia_dungeon admin progress reset PSEUDO test_facile
  • Suggestions joueurs en ligne, set/reset OK

6.3 - Anti-monopole

Un joueur avec 5+ completions cette semaine tente de join :

  • "Priorite aux nouveaux joueurs! Revenez dans Xs" pendant 1ere moitie
  • Peut join apres la moitie du timer
7
Recrutement : freeze + cliquable
▼

7.1 - Lancer

/arcadia_dungeon start test_facile
  • Broadcast [REJOINDRE (90s)] cliquable
  • Blindness + freeze pendant recrutement
  • Rappels cliquables a 60/30/10/5s

7.2 - Join + fin

/arcadia_dungeon join test_facile
  • J2 TP + freeze, J1 notifie
  • Fin recrutement: effets retires, "Les portes se ferment!"
  • Join apres → "Portes fermees!"
