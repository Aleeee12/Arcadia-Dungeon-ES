Arcadia Dungeon

Procedure de Test Finale v1.2.0
Configuration complete de donjons permanents en production
NeoForge 1.21.1 • par vyrriox

Progression globale

0/0
Remplacez les noms de donjons (crypt, mine, nether) par vos vrais donjons. Toutes les commandes sont pre-remplies et copiables en 1 clic.

Pre-requis

  • Serveur NeoForge 1.21.1 avec le JAR installe
  • OP 2+ pour /arcadia_dungeon admin
  • 2+ joueurs pour les tests multijoueurs
  • Zones de donjon construites sur la map
1
Demarrage serveur
▼

1.1 - Verification

  • Console: Arcadia Dungeon initializing
  • Dossier config/arcadia/dungeon/ cree
  • example_dungeon.json + weekly_config.json generes
2
Creation des donjons + zones
▼

2.1 - Creer

/arcadia_dungeon admin create crypt La Crypte des Damnes
/arcadia_dungeon admin create mine Les Mines Oubliees
/arcadia_dungeon admin create nether La Forteresse du Nether
/arcadia_dungeon admin list
  • 3 donjons crees et listes

2.2 - Renommer (si besoin)

/arcadia_dungeon admin rename crypt La Crypte Maudite
  • Donjon renomme OK

2.3 - Spawns

Se placer a l'entree de chaque donjon :

/arcadia_dungeon admin setspawn crypt
/arcadia_dungeon admin setspawn mine
/arcadia_dungeon admin setspawn nether
  • Spawns definis

2.4 - Zones (wand)

/arcadia_dungeon admin wand

Cliquer sur le donjon dans le chat, puis clic gauche coin 1, clic droit coin 2 :

/arcadia_dungeon admin wand_select mine
  • Wand recue, donjons cliquables
  • Zone crypt definie (dimensions en or)
  • Zone mine definie
  • Zone nether definie

Vagues de mobs (complet)

5.1 - Creer les vagues

/arcadia_dungeon admin wave setmobpos crypt 1 1

/arcadia_dungeon admin wave mob crypt 2 minecraft:husk 4
/arcadia_dungeon admin wave mob crypt 2 minecraft:stray 4
/arcadia_dungeon admin wave setmobpos crypt 2 0
/arcadia_dungeon admin wave setmobpos crypt 2 1
/arcadia_dungeon admin wave list crypt

/arcadia_dungeon admin wave glowing crypt 2 true 20

/arcadia_dungeon admin wave setmob crypt 1 1 damage 7
/arcadia_dungeon admin wave setmob crypt 2 0 health 40
/arcadia_dungeon admin wave setmob crypt 2 0 damage 8

5.4 - Equipement

  • Equip direct + copyequip OK

5.5 - Attributs mobs ATTR

Phases de boss
▼

7.1 - Creer les phases

/arcadia_dungeon admin boss phase add crypt gardien 2 0.6
/arcadia_dungeon admin boss phase set crypt gardien 2 damage 1.5
/arcadia_dungeon admin boss phase set crypt gardien 2 speed 1.3
/arcadia_dungeon admin boss phase set crypt gardien 2 invulnerable 3
/arcadia_dungeon admin boss phase set crypt gardien 2 message Le Gardien entre en rage!
/arcadia_dungeon admin boss phase add crypt gardien 3 0.25
/arcadia_dungeon admin boss phase set crypt gardien 3 damage 2.0
/arcadia_dungeon admin boss phase set crypt gardien 3 speed 1.5
/arcadia_dungeon admin boss phase set crypt gardien 3 action KILL_SUMMONS
/arcadia_dungeon admin boss phase set crypt gardien 3 message PHASE FINALE! Eliminez les serviteurs!
  • Phase 2 (60% HP) et Phase 3 (25% HP) creees

7.2 - Effets de phase + commandes

/arcadia_dungeon admin phaseeffect crypt gardien 2 minecraft:slowness 10 1
/arcadia_dungeon admin phaseeffect crypt gardien 3 minecraft:darkness 8 0
/arcadia_dungeon admin phasecommand crypt gardien 2 say Le Gardien rugit!
/arcadia_dungeon admin phasecommand crypt gardien 3 say Les morts se relevent!
  • Effets + commandes de phase configures

7.3 - Sbires de phase + attributs ATTR

/arcadia_dungeon admin boss phase summon crypt gardien 3 minecraft:zombie 4

11
Run complet du donjon
▼
Faire un run complet: vagues → mini-boss → vagues → boss final. Verifier chaque etape.

11.1 - Vague 1

  • Message "&cLes morts se levent..." affiche
  • Mobs equipes avec armes/armures
  • Attributs appliques (armor, etc.)
  • Noms personnalises affiches
  • Mobs ne drop RIEN (no-loot)
  • Scaling adaptatif: mobs plus forts si 2+ joueurs
  • Glowing apres 30s si pas tues

11.2 - Mini-boss (apres vague 1) NEW

  • "[Donjon] Un boss approche!" affiche
  • Mini-boss spawn avec bossbar jaune
  • Mini-boss invulnerable 5s au spawn
  • Mini-boss tue → vague 2 commence

11.3 - Vague 2

  • Delai 5s avant spawn
  • Message "&4Une horde approche!" affiche
  • Vague 2 terminee → "Toutes les vagues eliminees!"

11.4 - Boss final

  • Boss spawn equipe full netherite
  • Bossbar rouge affichee
  • Invulnerable 5s au spawn
  • Anti-phase-skip: impossible de one-shot

11.5 - Phases du boss

  • Phase 2 (60% HP): message + slowness + invuln 3s
  • Commande de phase executee ("Le Gardien rugit!")
  • Phase 3 (25% HP): darkness + sbires equipes + KILL_SUMMONS
  • Sbires avec attributs (armor 10, 40 HP)
  • "Eliminez les sbires!" affiche

11.6 - Victoire

  • Rewards boss + completion distribues
  • TP retour au point d'avant
  • Annonce victoire broadcast
  • Tous les mobs/sbires cleanup
  • Progression enregistree
