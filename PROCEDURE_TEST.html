Procedure de Test v1.1.0 - Donjons Permanents
Configuration de vrais donjons jouables en production
NeoForge 1.21.1 • par vyrriox

Progression globale

0/0
Cette procedure est prevue pour configurer et tester de VRAIS donjons permanents. Remplacez les noms/IDs par vos vrais donjons. Les commandes utilisent des exemples realistes.

Pre-requis

  • Serveur NeoForge 1.21.1 avec arcadia_dungeon-1.1.0.jar installe
  • Permission OP 2+ pour les commandes /arcadia_dungeon admin
  • 2+ joueurs disponibles pour les tests multijoueurs
  • Map du serveur prete avec les zones de donjon construites
  • Supprimer config/arcadia/dungeon/ si mise a jour depuis une version anterieure
1
Demarrage & Verification initiale
▼

1.1 - Logs serveur

  • Console affiche: Arcadia Dungeon initializing
  • Dossier config/arcadia/dungeon/ cree avec dungeons/, playerdata/
  • example_dungeon.json genere comme reference
  • weekly_config.json genere
2
Creation des donjons permanents
▼
Creez vos vrais donjons avec des noms definitifs. L'ID doit etre un mot unique sans espaces.

2.1 - Creer les donjons

Adaptez les IDs et noms a vos vrais donjons :

/arcadia_dungeon admin create crypt La Crypte des Damnes
/arcadia_dungeon admin create mine Les Mines Oubliees
/arcadia_dungeon admin create nether La Forteresse du Nether
/arcadia_dungeon admin list
  • Donjons crees, confirmes dans admin list

2.2 - Definir les spawns

Placez-vous a l'entree de chaque donjon :

/arcadia_dungeon admin setspawn crypt
/arcadia_dungeon admin setspawn mine
/arcadia_dungeon admin setspawn nether
  • Spawns definis pour chaque donjon

2.3 - Definir les zones (wand)

/arcadia_dungeon admin wand

Cliquez sur le donjon dans le chat, puis clic gauche coin 1, clic droit coin 2 pour chaque donjon :

  • Wand recue, liste donjons cliquable
  • Zone donjon 1 definie (dimensions affichees en or)
  • Zone donjon 2 definie
  • Zone donjon 3 definie
3
Configuration des settings
▼

3.1 - Settings principaux (par donjon)

Configurez chaque donjon selon sa difficulte :

/arcadia_dungeon admin settings crypt maxplayers 4
/arcadia_dungeon admin settings crypt minplayers 1
/arcadia_dungeon admin settings crypt maxdeaths 3
/arcadia_dungeon admin settings crypt recruitment 60
/arcadia_dungeon admin settings crypt timelimit 900
/arcadia_dungeon admin settings crypt pvp false
/arcadia_dungeon admin settings crypt teleportback true
/arcadia_dungeon admin settings mine maxplayers 6
/arcadia_dungeon admin settings mine minplayers 2
/arcadia_dungeon admin settings mine maxdeaths 5
/arcadia_dungeon admin settings mine timelimit 1200
/arcadia_dungeon admin settings nether maxplayers 4
/arcadia_dungeon admin settings nether maxdeaths 2
/arcadia_dungeon admin settings nether timelimit 600
  • Boolean: seuls true/false proposes en autocompletion
  • Settings sauvegardes (verifier avec admin info)

3.2 - Timer Warnings

/arcadia_dungeon admin timerwarnings crypt add 300
/arcadia_dungeon admin timerwarnings crypt add 120
/arcadia_dungeon admin timerwarnings crypt add 60
/arcadia_dungeon admin timerwarnings crypt add 30
/arcadia_dungeon admin timerwarnings crypt add 10
/arcadia_dungeon admin timerwarnings crypt list
  • Warnings ajoutes et listes correctement

3.3 - Cooldown + Disponibilite

/arcadia_dungeon admin cooldown crypt 1800
/arcadia_dungeon admin cooldown mine 3600
/arcadia_dungeon admin availability crypt 0 24
/arcadia_dungeon admin announce crypt Le donjon de la Crypte est ouvert!
  • Cooldown per-donjon fonctionne
  • Disponibilite horaire OK

3.4 - Blocage de commandes

Verifiez dans le JSON que blockedCommands contient les commandes a bloquer :

  • /tpa, /spawn, /home, /back bloques en donjon
  • /arcadia_dungeon abandon toujours autorise

3.5 - Anti-monopole

Dans le JSON: "antiMonopole": true, "antiMonopoleThreshold": 5

  • Anti-monopole configure

5
Vagues de mobs + equipement + attributs
▼

5.1 - Creer les vagues

/arcadia_dungeon admin wave add crypt 1
/arcadia_dungeon admin wave mob crypt 1 minecraft:zombie 6
/arcadia_dungeon admin wave mob crypt 1 minecraft:skeleton 3
/arcadia_dungeon admin wave setmobpos crypt 1 0
/arcadia_dungeon admin wave setmobpos crypt 1 1
/arcadia_dungeon admin wave add crypt 2
/arcadia_dungeon admin wave mob crypt 2 minecraft:husk 4
/arcadia_dungeon admin wave mob crypt 2 minecraft:stray 4
/arcadia_dungeon admin wave message crypt 1 Les morts se levent...
/arcadia_dungeon admin wave message crypt 2 Une horde approche!
/arcadia_dungeon admin wave list crypt
  • Vagues creees avec mobs multiples
  • Positions definies pour chaque type de mob
  • Messages personnalises

5.2 - Equipement des mobs de vague

/arcadia_dungeon admin wave equip crypt 1 0 mainhand minecraft:iron_sword
/arcadia_dungeon admin wave equip crypt 1 0 helmet minecraft:iron_helmet
/arcadia_dungeon admin wave equip crypt 1 1 mainhand minecraft:bow
/arcadia_dungeon admin wave copyequip crypt 2 0
  • Equipement set par commande directe
  • Copyequip copie le stuff du joueur

5.3 - Attributs des mobs de vague ATTR

/arcadia_dungeon admin wave attribute set crypt 1 0 minecraft:generic.armor 8
/arcadia_dungeon admin wave attribute set crypt 1 0 minecraft:generic.knockback_resistance 0.5
/arcadia_dungeon admin wave attribute set crypt 1 1 minecraft:generic.follow_range 48
/arcadia_dungeon admin wave attribute list crypt 1 0
/arcadia_dungeon admin wave attribute remove crypt 1 0 minecraft:generic.knockback_resistance
/arcadia_dungeon admin wave attribute list crypt 1 0
  • Autocompletion des attributs (tous les attributs vanilla sugeres)
  • Attributs set + confirme en vert
  • List affiche les attributs configures
  • Remove supprime l'attribut, list confirme

5.4 - Glowing apres delai

Dans le JSON de la vague: "glowingAfterDelay": true, "glowingDelaySeconds": 30

  • Mobs brillent apres le delai si pas tues

/arcadia_dungeon admin boss add crypt spectre minecraft:wither_skeleton 100 15

Dans le JSON, mettre sur le 2e boss: "optional": true, "spawnChance": 0.3

  • Boss optionnel configure (30% chance)
7
Phases de boss : effets, sbires, commandes
▼

7.1 - Creer les phases

/arcadia_dungeon admin boss phase add crypt gardien 2 0.6
/arcadia_dungeon admin boss phase set crypt gardien 2 damage 1.5
/arcadia_dungeon admin boss phase set crypt gardien 2 speed 1.3
/arcadia_dungeon admin boss phase set crypt gardien 2 invulnerable 3
/arcadia_dungeon admin boss phase set crypt gardien 2 message Le Gardien entre en rage!
/arcadia_dungeon admin boss phase add crypt gardien 3 0.25
/arcadia_dungeon admin boss phase set crypt gardien 3 damage 2.0
/arcadia_dungeon admin boss phase set crypt gardien 3 speed 1.5
/arcadia_dungeon admin boss phase set crypt gardien 3 action KILL_SUMMONS
/arcadia_dungeon admin boss phase set crypt gardien 3 message PHASE FINALE! Eliminez les serviteurs!
  • Phase 2 a 60% HP, Phase 3 a 25% HP
  • Invulnerabilite + multiplicateurs configures

7.2 - Effets de phase sur les joueurs

/arcadia_dungeon admin phaseeffect crypt gardien 2 minecraft:slowness 10 1
/arcadia_dungeon admin phaseeffect crypt gardien 3 minecraft:darkness 8 0
/arcadia_dungeon admin phaseeffect crypt gardien 3 minecraft:weakness 10 0
  • Effets configures par phase

7.3 - Commandes de phase

/arcadia_dungeon admin phasecommand crypt gardien 2 say Le Gardien rugit de colere!
/arcadia_dungeon admin phasecommand crypt gardien 3 say Les morts se relevent pour proteger leur maitre!
  • Commandes associees aux phases

7.4 - Sbires de phase + attributs ATTR

/arcadia_dungeon admin boss phase summon crypt gardien 3 minecraft:zombie 4
/arcadia_dungeon admin boss phase summonequip crypt gardien 3 0 mainhand minecraft:iron_sword
/arcadia_dungeon admin boss phase summonequip crypt gardien 3 0 helmet minecraft:chainmail_helmet
/arcadia_dungeon admin boss phase summonattribute set crypt gardien 3 0 minecraft:generic.armor 10
/arcadia_dungeon admin boss phase summonattribute set crypt gardien 3 0 minecraft:generic.max_health 40
/arcadia_dungeon admin boss phase summonattribute set crypt gardien 3 0 minecraft:generic.attack_damage 8
/arcadia_dungeon admin boss phase summonattribute list crypt gardien 3 0
  • Sbires equipes + attributs set
  • List affiche les attributs du sbire

7.5 - Remove attribut sbire

/arcadia_dungeon admin boss phase summonattribute remove crypt gardien 3 0 minecraft:generic.attack_damage
/arcadia_dungeon admin boss phase summonattribute list crypt gardien 3 0
  • Attribut supprime et list confirme

Test de recrutement
▼

9.1 - Lancer le donjon

/arcadia_dungeon start crypt
  • Broadcast serveur [REJOINDRE] cliquable avec timer
  • Joueur initiateur TP + freeze + blindness
  • Rappels cliquables a 30s/10s/5s

9.2 - Join et fermeture

/arcadia_dungeon join crypt
  • J2 TP dans le donjon + freeze + notification J1
  • Fin recrutement: effets retires, "Les portes se ferment!"
  • Tentative join apres → "Les portes du donjon sont fermees!"
  • Mobs existants dans la zone cleares au demarrage
10
Gameplay complet (run entier)
▼
Faites un run complet du donjon pour verifier tous les systemes ensemble.

10.1 - Vagues

  • Mobs spawn equipes avec armes/armures
  • Attributs personnalises appliques (armor, speed, etc.)
  • Mobs ne drop RIEN (no-loot)
  • Messages de vague affiches
  • Vagues enchainees correctement
  • Glowing apres delai si configure

10.2 - Boss fight

  • Boss spawn equipe, barre de boss affichee
  • Attributs boss appliques (armor, knockback_resistance, etc.)
  • Analyse d'armure joueurs (log console)
  • Anti-phase-skip: impossible de one-shot le boss

10.3 - Phases de boss

  • Phase 2 se declenche a 60% HP
  • Message de phase affiche
  • Effets de potion appliques aux joueurs
  • Invulnerabilite pendant la duree configuree
  • Commandes de phase executees
  • Phase 3: sbires equipes + attributs appliques
  • "Eliminez les sbires!" si action = KILL_SUMMONS

10.4 - Victoire

  • Rewards distribues (boss + completion)
  • TP retour au point d'avant
  • Annonce victoire avec noms des joueurs
  • Tous les mobs cleanup
  • Progression enregistree
11
Systeme de vies + securite
▼

11.1 - Systeme de vies

  • Mort → respawn au spawn du donjon + "X vie(s) restante(s)"
  • Derniere mort → exclusion du donjon + notification equipe
  • Pas de TP retour parasite apres max deaths

11.2 - Containment

  • Joueur sort de la zone → TP retour au spawn donjon
  • Mobs de vague qui sortent → TP retour
  • Admin en spectator PAS teleporte
  • Joueur non-participant dans zone → TP dehors

11.3 - Anti-PVP + commandes

  • Anti-PVP entre coequipiers actif
  • Commandes bloquees pendant le donjon
  • /arcadia_dungeon abandon toujours autorise
12
Leave / Abandon / Echec / Timer
▼

12.1 - Abandon et notifications

/arcadia_dungeon abandon
  • "X a quitte le donjon! (Y restant(s))"
  • Dernier quitte → echec broadcast avec noms des joueurs
  • Deconnexion → joueur retire + boss bar nettoyee

12.2 - Timeout

Mettre un timelimit court (60s) pour tester rapidement :

/arcadia_dungeon admin settings crypt timelimit 60
  • Warnings aux timers configures (meme avec lag)
  • "Temps ecoule!" + echec broadcast avec noms
  • Remettre le vrai timelimit apres le test
